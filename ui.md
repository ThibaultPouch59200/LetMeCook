Factorio UI style brief for code generation. Background colors: main background is #1a1a1a, panel background is #2a2a2a or #1e1e1e, borders are #4a4a3a which is a dark metallic brown, main accent is #f0a030 which is warm industrial orange, secondary accent is #c8a020 which is dull gold, main text is #e0d8c0 which is off-white or parchment, secondary text is #a09880 which is beige gray, success or positive color is #60c840 which is factory green, danger or alert is #e03020 which is bright red, button background is #3a3020 and on hover becomes #5a4a30. Typography should use monospace or pixel-like fonts such as Share Tech Mono, Courier New, or Orbitron. All titles and labels should be uppercase. Text should be slightly yellowed, never pure white. Borders should be thick at 2 to 3px, style solid or double, corners should not be rounded so border-radius should be 0 or 2px maximum. Panels should have a simulated bevel effect using inset box-shadow with values inset 2px 2px 0 #6a6040 and inset -2px -2px 0 #1a1408. Panel backgrounds should have a subtle noise texture or a repeating-linear-gradient to simulate it. Buttons have a dark background of #3a3020 with border #6a5a30, on hover the background becomes #5a4a20 with a slight orange highlight, on click use inset shadow for a pressed effect, add an icon on the left if possible using industrial sprites or emojis, the overall button style is military and industrial not Material Design. Icons should be pixel-art or industrial style SVGs. Progress bars should have a dark background with orange or green fill and no smooth animation, they should feel stepped or abrupt. Inventory slots should be a grid of 48x48 squares with very dark background, thin border, and orange highlight on hover. Layout uses floating panels with a title bar at the top in a slightly lighter background. Panel headers use background #3a3020 with uppercase text and an orange line separator. Use custom dark scrollbars if possible. Avoid soft shadows, everything should feel solid, mechanical, and functional. The overall vibe is an industrial machine interface from the 90s projected into a dystopian future, functional before aesthetic, every pixel has a purpose.
